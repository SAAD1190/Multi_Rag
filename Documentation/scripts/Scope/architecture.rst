System Architecture

---------------------

**Overview**:
1. **Retrieval Layer**: Handles document or data retrieval in multiple languages.
2. **Generation Layer**: Processes the retrieved content to provide multilingual responses.
3. **Interface**: User-friendly interactive app built using Streamlit.
